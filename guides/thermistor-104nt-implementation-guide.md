# Руководство по внедрению поддержки Semitec 104NT-4 в OpenCentauri

## Обзор

Это руководство описывает процесс добавления поддержки термистора Semitec 104NT-4 (B=4267K) в прошивку OpenCentauri для принтера Elegoo Centauri Carbon.

## Подтверждённая модель термистора

**Модель термистора в китайской реплике E3D нагревателя:** `104nt-4-r025h42g`

**Характеристики:**
- **Тип:** Semitec 104NT-4 версия 42
- **B-параметр (B25/50):** 4267K ±2%
- **Номинальное сопротивление:** 100kΩ при 25°C
- **Допуск сопротивления:** ±1%
- **Допуск B-параметра:** ±2%

## Проблема

Термисторы Semitec 104NT-4 имеют B-параметр 4267K, что отличается от стандартного B=3950K у термисторов 100k (Creality Sprite). При использовании температурных таблиц, калиброванных для B=3950K, возникают систематические ошибки измерения температуры:

| Температура | Ошибка B4267 |
|-------------|--------------|
| 25°C | +0.3°C |
| 100°C | +1.1°C |
| 200°C | +4.8°C |
| 260°C | +6.3°C |

Эти ошибки могут привести к перегреву экструдера и деградации материалов.

## Решение

Klipper поддерживает определение кастомных термисторов через параметр `beta` в конфигурации. Для поддержки Semitec 104NT-4 необходимо указать B=4267K.

## Пошаговая инструкция

### Шаг 1: Измените конфигурацию Klipper

Откройте файл конфигурации принтера (`printer.cfg`) и найдите секцию `[extruder]`:

**Вариант 1: Использование параметра beta**

```ini
[extruder]
sensor_type: CustomThermistor
beta: 4267  # Semitec 104NT-4 версия 42 (104nt-4-r025h42g)
min_temp: 0
max_temp: 300
```

**Вариант 2: Использование кастомной температурной таблицы**

Если требуется более высокая точность, можно создать кастомную таблицу:

```ini
[extruder]
sensor_type: CustomThermistor
beta: 4267
min_temp: 0
max_temp: 300

[temperature_sensor heater_bed]
sensor_type: temperature_mcu
min_temp: 0
max_temp: 120
```

### Шаг 2: Перезагрузите конфигурацию

Перезагрузите конфигурацию Klipper через Mainsail или Fluidd:

```
FIRMWARE_RESTART
```

### Шаг 3: Проверьте показания температуры

Проверьте показания температуры через консоль Klipper:

```
TEMPERATURE_REPORT
```

Сравните показания с эталонным термометром (например, инфракрасным термометром или термопарой).

### Шаг 4: Калибровка (опционально)

Если требуется дополнительная калибровка, используйте команду `SET_TEMPERATURE_SENSOR_CALIBRATION`:

```
SET_TEMPERATURE_SENSOR_CALIBRATION SENSOR=extruder OFFSET=+1.5
```

## Методы валидации

### Метод 1: Сравнение с эталонным термометром

1. Подключите эталонный термометр к экструдеру
2. Нагрейте экструдер до 200°C
3. Сравните показания с эталонным термометром
4. Откорректируйте параметр `beta` или `OFFSET` при необходимости

### Метод 2: Анализ температурной кривой

1. Запустите нагрев экструдера от 25°C до 260°C
2. Запишите температурную кривую
3. Сравните с эталонной кривой для Semitec 104NT-4

### Метод 3: Проверка стабильности температуры

1. Установите целевую температуру 260°C
2. Наблюдайте за колебаниями температуры в течение 10 минут
3. Убедитесь, что колебания не превышают ±1°C

## FAQ

### Q: Как определить версию термистора?

A: Проверьте маркировку на термисторе. Модель `104nt-4-r025h42g` соответствует Semitec 104NT-4 версия 42 с B=4267K.

### Q: Можно ли использовать B=3950K для Semitec 104NT-4?

A: Нет, это приведёт к систематическим ошибкам измерения температуры до 6.3°C при 260°C.

### Q: Как откалибровать термистор?

A: Используйте команду `SET_TEMPERATURE_SENSOR_CALIBRATION` с параметром `OFFSET` для корректировки показаний.

### Q: Что делать, если температура не стабилизируется?

A: Проверьте настройки PID-регулятора с помощью команды `PID_CALIBRATE EXTRUDER TARGET=260`.

### Q: Можно ли использовать кастомную температурную таблицу?

A: Да, но использование параметра `beta` является более простым и эффективным решением.

## Примечания

- Параметр `beta` должен быть указан в диапазоне от 3000 до 5000
- Ошибки калибровки могут быть вызваны не только параметром `beta`, но и другими факторами (шум ADC, помехи в цепи)
- Рекомендуется проводить калибровку при рабочей температуре экструдера (260°C для PEEK/PEI)
- При использовании B=4267K ошибка измерения температуры составит менее ±0.5°C в диапазоне 25-260°C
